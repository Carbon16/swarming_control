# Security

**Please be aware that this tool is not intended to run on a public network, and does not include security features such as authentication or encryption out of the box. It is designed for use on a local network, hotspot, or behind a secure firewall.**

## Reporting a Vulnerability

Please email lgs1g24@soton.ac.uk as soon as possible. Alternatively, you can use the github security advisory system to report a vulnerability. We will respond as quickly as possible and work with you to resolve the issue.

## Supported Versions

- Version 0.1.0 (current)
